Which shell feels right for the Teacher Dashboard?

A — Icon Sidebar

Compact, always-visible icon nav on the left. Hover or click to see labels. Maximizes content area. Used by Linear, Figma, Vercel.

B — Top Navigation Bar

Horizontal nav across the top. All content below. Simple and familiar. Used by most classic admin dashboards and Google products.

C — Wide Sidebar with Labels

Full-width sidebar showing icon + text for every nav item. Class selector built in. Used by Stripe, Notion, Linear (expanded). Best for discoverability.
